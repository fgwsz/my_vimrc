## 一、基础环境与兼容性
*（保障 Vim 正常运行，屏蔽 Vi 兼容模式，定义底层行为）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **Vi 兼容性** | `set nocompatible` | 关闭 Vi 兼容模式，启用 Vim 高级特性（必须放在最前面）。 |
| **文件载入** | `set autoread` | 当文件在外部被修改时，自动重新读取，保持同步。 |
| **临时文件管理** | `set nobackup`<br>`set noswapfile`<br>`set nowritebackup`<br>`set noundofile` | **完全禁用**备份文件（`~`）、交换文件（`.swp`）、写入备份和撤销文件（`.un~`），保持工作目录干净。 |
| **退格键（Backspace）** | `set backspace=indent,eol,start` | 允许在插入模式下删除缩进、跨行删除和删除行首之前的字符。 |
| **错误提示** | `set noerrorbells`<br>`set visualbell`（关闭） | 关闭错误提示音和屏幕闪烁，安静编辑。 |
| **退出确认** | `set noconfirm` | 关闭退出时的繁琐确认提示（谨慎使用，强制退出需 `:qa!`）。 |
| **粘贴模式** | `set nopaste` | 默认关闭 Paste 模式，保持自动缩进正常工作。 |

## 二、文件编码与跨平台字符集
*（解决中文乱码、多编码文档识别问题）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **默认编码** | `set encoding=utf-8` | 设置 Vim 内部默认使用 UTF-8 编码。 |
| **自动编码检测** | `set fileencodings=ucs-bom,utf-8,cp936,gb18030,big5,euc-jp,euc-kr,latin1` | 按**从左到右**的顺序自动探测文件编码，完美支持简体中文（GBK/GB18030）、繁体中文（Big5）和日韩编码。 |
| **命令行补全增强** | `set wildmenu` | 命令模式下按 Tab 呼出可视化补全菜单，方向键切换选项。 |

## 三、界面视觉与编辑辅助
*（行号、高亮、缩进、特殊字符显示等视觉体验）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **行号显示** | `set number` | 显示绝对行号（左侧）。 |
| **长行处理** | `set nowrap`<br>`set sidescroll=1` | 取消自动折行，超长行水平滚动，且每次滚动 **1 个字符**，非常平滑。 |
| **光标定位** | `set scrolloff=999` | 滚动时始终**将光标所在行置于屏幕正中央**，极大提升视觉焦点稳定性。 |
| **代码列宽警示** | `set colorcolumn=81` | 在第 81 列显示一条灰色竖线，提醒代码列宽限制（如 PEP8）。 |
| **自动换行控制** | `set textwidth=0` | 禁止 Vim 在输入时自动插入换行符（配合 `colorcolumn` 仅作视觉提示）。 |
| **语法高亮** | `syntax on` | 开启语法高亮（关键字、字符串、注释等）。 |
| **搜索高亮** | `set hlsearch`<br>`set incsearch` | 搜索结果高亮，且**键入时即时增量高亮**（按 `<Esc>` 可一键清除高亮）。 |
| **括号匹配** | `set showmatch` | 光标移动到括号时，短暂高亮与之匹配的另一个括号。 |
| **不可见字符显示** | `set list`<br>`set listchars=tab:>―,space:·,eol:↓` | 显示 Tab（`>―`）、空格（`·`）和换行符（`↓`），便于排查空格与缩进问题。 |
| **配色方案** | `set background=dark`<br>`colorscheme industry` | 适配深色背景，选用 Industry 配色方案。 |
| **缩进设置（空格化）** | `set tabstop=4`<br>`set softtabstop=4`<br>`set shiftwidth=4`<br>`set expandtab` | Tab 键显示宽度为 4，缩进宽度为 4，且**键入 Tab 自动转为 4 个空格**（符合多数现代编码规范）。 |

## 四、标签栏（Tabline）定制
*（顶部标签栏的视觉增强与信息展示）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **始终显示标签栏** | `set showtabline=2` | 无论是否打开多标签，始终显示标签栏。 |
| **标签页名动态生成** | `TabName()` 函数 | 自动显示文件名；若为 Netrw 显示 "Netrw"，终端显示 "Terminal"，无文件名显示 "No Name"。 |
| **标签页修改状态** | `TabModified()` 函数 | 若该标签页内有未保存修改，标签名旁显示 **`＊`** 号。 |
| **窗口数提示** | `TabLine()` 函数 | 若当前标签页内含有多个分屏窗口，在标签名后显示 `(窗口数)`。 |
| **标签高亮配色** | `Highlight`（GUI/CTerm） | 区分**当前选中标签**（蓝底白字）和**非选中标签**（灰底暗字），修改状态用红底白字醒目提示。 |

## 五、状态栏（Statusline）定制
*（底部状态栏的完全自定义，实时反馈编辑状态）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **始终显示状态栏** | `set laststatus=2` | 始终在底部显示状态栏。 |
| **动态内容展示** | `InitStatusLine()` 函数 | 从左到右依次展示：<br>1. **当前模式**（Normal/Insert/Visual 等）<br>2. **文件路径**（相对/绝对）<br>3. **只读标识**（`RO`）<br>4. **修改标识**（`+`）<br>5. **文件类型**（如 `vim`/`python`）<br>6. **操作系统格式**（Unix/DOS）<br>7. **文件编码**（UTF-8 等）<br>8. **光标位置**（列:行/总行数 百分比）<br>9. **折行状态**（Wrap/NoWrap） |
| **状态栏配色** | `User1~8` 高亮组 | 不同信息段使用不同背景色（深绿/蓝/灰/红等），信息层级一目了然。 |
| **底部辅助信息** | `set ruler`<br>`set showmode`<br>`set showcmd` | 额外显示光标位置、当前模式，以及正在键入的按键序列。 |

## 七、Netrw 文件管理器增强
*（内置 `:Explore` 的定制与跨平台修复）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **显示模式** | `let g:netrw_banner=1`<br>`let g:netrw_liststyle=1` | 显示顶部横幅，列表样式为 **Long**（显示大小/日期）。 |
| **排序规则** | `let g:netrw_sort_by='name'` | 按文件名排序。 |

## 八、内置终端（Terminal）集成与控制
*（Vim 8.0+ 内置终端的深度适配）*

| 功能分类 | 具体配置/命令 | 作用说明 |
| :--- | :--- | :--- |
| **默认 Shell** | `set shell=powershell`（Win）<br>`set shell=/bin/bash`（Linux） | 根据操作系统切换终端底层 Shell。 |
| **剪贴板互通** | `set clipboard^=unnamed,unnamedplus` | Vim 的复制粘贴直接与系统剪贴板打通（`y` 复制到系统，`p` 粘贴自系统）。 |
| **终端退出插入模式** | `<C-n>`（`tnoremap`） | 在终端窗口中按 `Ctrl-n` 回到普通模式（方便复制终端输出）。 |
| **终端粘贴** | `<C-p>`（`tnoremap`） | 终端中按 `Ctrl-p` 粘贴系统剪贴板内容（规避 `Ctrl-v` 与列选择冲突）。 |
| **路径提取** | `GetTerminalCurrentLinePath()` 函数 | 智能解析终端光标所在行，提取当前工作路径（支持 PowerShell 的 `PS X:\path>` 和 Bash 的 `user@host:path$` 格式）。 |

## 九、快捷键映射总览（含完整模式标注）

> **图例说明**：  
> **N** = 普通模式（Normal）  
> **I** = 插入模式（Insert）  
> **V** = 可视模式（Visual）  
> **T** = 终端模式（Terminal）  
> **C** = 命令行模式（Command-line）

### 9.1 核心文件操作（保存/关闭/全选/撤销）

| 快捷键 | 适用模式 | 功能说明 |
| :--- | :--- | :--- |
| **`Ctrl-s`** | **N / I / V** | **保存**当前文件（`w!`）。插入模式下保存后自动回到插入位置。 |
| **`Ctrl-q`** | **N / I / V / T** | **强制关闭**当前窗口/分屏（`q!`）。终端中会先切到普通模式再退出。 |
| **`Ctrl-a`** | **N / I / V** | **全选**当前缓冲区全部内容（`gg0vG$`）。 |
| **`Y`** | **N / V** | **复制当前行**（不含换行符），即 `0y$`。 |
| **`Ctrl-d`** | **N / I** | **复制当前行到下一行**（`yyp`），插入模式下自动回到插入状态。 |
| **`Ctrl-z`** | **N / I** | **撤销**（`u`）。 |
| **`Ctrl-y`** | **N / I** | **重做**（`Ctrl-r`）。 |
| **`Esc`** | **N / I / V** | **关闭搜索高亮**（`:nohlsearch`），按一次即可消除黄色高亮。 |

### 9.2 文本编辑与格式化辅助（生产力工具）

| 快捷键/命令 | 适用模式 | 功能说明 |
| :--- | :--- | :--- |
| **`Alt-c`** | **N / I / V** | **一键替换**全文中文标点为英文标点（包含 `！`→`!`，`。`→`.`，`“”`→`""` 等全套映射）。 |
| **`o`**（重写） | **N** | **在当前行下方新增空白行**，并保持光标位置（保护剪贴板）。 |
| **`O`**（重写） | **N** | **在当前行上方新增空白行**，并保持光标位置（保护剪贴板）。 |
| **`>>`**（重写） | **N** | **增加缩进**（4空格），支持空行，**光标水平位置不变**。 |
| **`<<`**（重写） | **N** | **减少缩进**（删4空格），**光标水平位置不变**（原生会跳回行首）。 |
| **`an`** | **V** | 删除选中区域的**所有换行符**（行合并），光标移到行首。 |
| **`al`** | **V** | 删除选中区域的**所有空行**（含仅空格的行）。 |
| **`as`** | **V** | 删除选中区域的**所有空白字符**（空格/Tab等）。 |
| **`at`** | **V** | 删除选中区域**所有行尾多余空格**。 |
| **`ah`** | **V** | 删除选中区域**所有行首多余空格**。 |
| **`F3`** | **V** | **统计选中区域非空白字符数量**（完美支持中文等多字节字符）。 |
| **`qa` ~ `qz`** <br>**`q0` ~ `q9`** | **N** | **清空寄存器后再录制宏**（共36个寄存器），避免旧宏意外叠加。 |

## 十、自动化事件（Autocmd）
*（Vim 自动触发的后台任务）*

| 触发事件 | 自动执行的动作 | 作用说明 |
| :--- | :--- | :--- |
| **`WinEnter *`** | `SetSpecialCharactersDarkGrey()` | 进入任何窗口时，将 Tab/空格/换行符的显示颜色统一设为**深灰色**（避免原配色下看不清）。 |
| **`BufEnter *`** | `LCdCurrentPath()` | **切换到任何缓冲区时**，自动将当前窗口的工作目录（`lcd`）切换为该文件所在的父目录，方便相对路径操作。 |
| **`BufWrite *`** | `:nohlsearch` | 每次保存文件后，自动**清除搜索高亮**（省去手动按 Esc）。 |
| **`VimEnter *`** | `InitStatusLine()` | Vim 完全启动后，初始化状态栏（修复首次启动时折行状态不显示的问题）。 |
| **`BufWritePost $MYVIMRC`** | `source $MYVIMRC` | 修改并保存本 `vimrc` 文件后，**立即自动重新加载**生效，无需重启 Vim。 |

## 十二、附注：已注释的备用功能

原始 `vimrc` 中尚有几段被注释掉的配置，它们目前**未生效**，但您可以根据需要取消注释以启用。以下列出这些功能及其用途：

| 配置块（注释状态） | 功能描述 | 启用方法 |
| :--- | :--- | :--- |
| **光标行列高亮**（`set cursorline` / `set cursorcolumn`） | 高亮显示光标当前所在行和列，有助于快速定位。配套有配色定义（`highlight CursorLine` 等）。 | 取消 `set cursorline` 和 `set cursorcolumn` 的注释，同时取消其下方的 `highlight` 配置块。 |
| **终端光标形状跟随模式**（`&t_SI` 等） | 在终端 Vim 中，根据当前模式（普通/插入/替换）自动切换光标形状（方块/竖线/下划线），提升视觉反馈。 | 取消 `if has('win32')` 内部的 `let &t_SI` 等三行注释。**注意**：该功能在某些终端下可能存在显示 Bug（原文已警示）。 |
| **Netrw 自动启动**（`autocmd VimEnter * :Vexplore`） | 启动 Vim 时自动在左侧打开 Netrw 文件浏览器，方便直接浏览文件。 | 取消 `augroup ProjectDrawer` 及其内部的 `autocmd` 注释。 |
| **Netrw 分屏方式**（`g:netrw_browse_split`） | 控制 Netrw 中打开文件时采用何种分屏方式（水平/垂直/新标签等）。 | 取消 `let g:netrw_browse_split=4` 的注释（默认值为 0，您可调整数字）。 |
| **Netrw 窗口宽度**（`g:netrw_winsize`） | 设置 Netrw 分屏窗口占据整个 Vim 窗口的百分比（如 25%）。 | 取消 `let g:netrw_winsize=25` 的注释并调整数值。 |
| **Netrw 排序方式**（`g:netrw_sort_by` / `g:netrw_sort_direction`） | 按时间排序并降序排列文件。 | 取消这两行的注释（会覆盖当前的按名称排序）。 |
| **Netrw 打开文件操作（`x` 键）** | 原生 `x` 键在部分版本中存在跨平台问题，本配置已用 `gx` 替代。如果您的 Vim 版本已修复，可取消注释并恢复 `x` 的默认行为。 | （此配置未直接注释，但您可自行映射 `x` 为 `:call OpenFileWithSystemExplorer()` 等。） |

> **提示**：上述被注释的功能均保留在原始 `vimrc` 中，您只需删除对应的 `"` 注释符号即可启用。启用前请确认您的终端或 Vim 版本兼容性。
